# beeins-audio

Pre-rendered pronunciation clips for [beeins](https://github.com/universeawaits/beeins),
served to the app via jsDelivr CDN.

- `audio/de/` — German (Piper `de_DE-thorsten-medium`)
- `audio/tr/` — Turkish (Piper `tr_TR-dfki-medium`)

Each file is `<cyrb53(text)>.m4a` (AAC). Generated offline; MIT-licensed Piper voices (Rhasspy).
